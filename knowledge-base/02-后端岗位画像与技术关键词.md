# 后端岗位画像与技术关键词

## 一、Java 后端开发

### 核心技术栈关键词

| 层级 | 技术 | 掌握深度要求 |
|------|------|-------------|
| 语言基础 | Java 8/11/17、JVM 内存模型、GC 调优、并发编程(JUC) | 熟练 → 深入理解 |
| 框架 | Spring Boot、Spring Cloud、MyBatis/MyBatis-Plus | 熟练使用 |
| 微服务 | Spring Cloud Alibaba(Nacos/Sentinel/Seata)、Dubbo、gRPC | 有实战经验 |
| 数据库 | MySQL(索引优化/分库分表)、Redis(缓存策略/集群)、MongoDB | 深入理解 |
| 中间件 | RocketMQ/Kafka(消息可靠性/顺序消息)、Elasticsearch | 有实战经验 |
| 运维 | Docker、Kubernetes、Jenkins/GitLab CI、Prometheus+Grafana | 了解 → 熟练 |

### 面试高频考点

- HashMap 底层原理（数组+链表+红黑树，扩容机制）
- JVM 内存区域与 GC 算法
- MySQL 索引原理（B+Tree）、慢查询优化
- Redis 缓存穿透/击穿/雪崩及解决方案
- 分布式事务（2PC/TCC/Saga/消息最终一致性）
- 分布式锁（Redis/ZooKeeper/Redisson）
- Spring IOC/AOP 原理
- 线程池参数配置与拒绝策略

## 五、后端架构师

### 核心能力模型

| 能力维度 | 具体要求 |
|----------|---------|
| 系统设计 | 分布式系统理论(CAP/BASE)、高可用架构、数据一致性方案 |
| 架构演进 | 单体 → SOA → 微服务 → Serverless 的演进路径与判断依据 |
| 数据架构 | 分库分表策略、数据湖/数据仓库、实时/离线计算 |
| 安全架构 | 认证授权(OAuth2/JWT)、数据加密、安全审计 |
| 技术选型 | 能够基于业务场景做出合理的技术选型决策并阐述理由 |
| 团队建设 | 技术规范制定、Code Review 体系、技术梯队培养 |

### 架构师面试高频考点

- 如何设计一个秒杀系统
- 如何设计一个即时通讯系统
- 分布式 ID 生成方案（雪花算法）
- 限流/熔断/降级策略
- 数据库分库分表（水平拆分策略）
- 大文件存储与分发（OSS+CDN）
- 多活架构设计
- 领域驱动设计(DDD)实践
